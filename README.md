How to use git to put new results into the overleaf project:

1. `cd git_overleaf`
1. `git pull`
1. `../update_results.sh`
1. `git add --all`
1. `git commit -m "updated results"`
1. `git status`   (make sure all is well)
1. `git push`
1. `cd ..`

Then in the overleaf browser, recompile the project.
